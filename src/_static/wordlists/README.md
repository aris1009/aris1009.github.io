# Wordlists

## eff_large_wordlist-v1.txt

The EFF Large Wordlist for Passphrases (7,776 words), distributed under CC-BY-3.0 by the Electronic Frontier Foundation.

- Source: https://www.eff.org/dice
- Original file: https://www.eff.org/files/2016/07/18/eff_large_wordlist.txt
- Version: v1 (2016-07-18 EFF publication)
- SHA-256: `addd35536511597a02fa0a9ff1e5284677b8883b83e986e43f15a3db996b903e`

The file is bundled with this site so you can audit the exact list used by the in-browser diceware generator at `/tools/diceware/`. Nothing is fetched from a third-party origin at runtime.

The hash above is verified at build time by `scripts/verify-wordlists.js` and at run time by the client-side loader against a pinned constant.
